*[GGRS]: Good Game Rollback System - The original library this project is forked from
*[GGPO]: Good Game Peace Out - The original C++ rollback netcode library
*[P2P]: Peer-to-Peer - Direct communication between game clients without a central server
*[RTT]: Round-Trip Time - Time for a message to travel to a peer and back
*[FPS]: Frames Per Second - The rate at which game frames are rendered
*[UDP]: User Datagram Protocol - A connectionless network protocol used for low-latency communication
*[TCP]: Transmission Control Protocol - A reliable, connection-oriented network protocol
*[API]: Application Programming Interface
*[ABI]: Application Binary Interface
*[TLA+]: Temporal Logic of Actions - A formal specification language
*[RNG]: Random Number Generator
*[RLE]: Run-Length Encoding - A compression algorithm
*[CRC]: Cyclic Redundancy Check - An error-detecting code
*[WASM]: WebAssembly - A binary instruction format for web browsers
*[CLI]: Command Line Interface
*[CI]: Continuous Integration
*[CD]: Continuous Deployment
